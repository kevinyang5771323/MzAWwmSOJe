# Python在线阅读推荐系统 python965

## 项目概述

本项目是一个基于Python语言的在线阅读推荐系统，采用前后端分离的架构设计，前端使用Vue.js框架，后端采用Django框架，并使用MySQL作为数据库管理系统。

## 技术栈

- 后端：Python + Django
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

系统主要分为以下功能模块：

- 用户模块
- 管理员模块
- 阅读推荐模块
- 文档管理模块

## 系统角色

- 管理员
- 普通用户

## 运行环境

- 服务器：符合Django和MySQL运行要求的Linux/Windows服务器
- 客户端：任何支持现代Web浏览器的设备

## 部署步骤

1. 准备符合要求的运行环境
2. 安装必要的软件和依赖
3. 配置数据库MySQL
4. 部署后端服务
5. 部署前端服务

## 目录结构

```
├── backend                # 后端代码目录
│   ├── apps               # 系统应用目录
│   ├── settings.py        # 系统配置文件
│   └── ...
├── frontend               # 前端代码目录
│   ├── assets             # 资源文件目录
│   ├── components         # Vue组件目录
│   └── ...
└── ...
```
